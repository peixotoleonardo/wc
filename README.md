# WC

Este é um projeto de estudo para aprender novas linguagens de programação de forma
prática.

## O que é wc?

O `wc` é um comando presente no Unix que permite contar:

- bytes (-c, ou --bytes)
- caracteres (-m, ou --chars)
- linhas (-l, ou --lines)
- palavras (-w, ou --words)

em um ou mais arquivos.
